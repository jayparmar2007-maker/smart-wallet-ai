# 📊 Personal Finance / Expense Tracker

A full-stack **Personal Finance Management Web App** built using **Node.js, Express, HTML, CSS, and JavaScript**.  
This application allows users to manage income, track expenses, and calculate profit with visual insights.

---

## 🚀 Features

### 🔐 Authentication
- User Registration
- User Login
- Separate user data storage

---

### 💰 Financial Management
- Add Income
- Add Expenses
- Delete Expenses
- (Optional) Edit Expenses

---

### 📊 Profit System
- Automatically calculates:
  - Total Income
  - Total Expenses
  - Profit = Income − Expenses

---

### 📅 Monthly Reports
- View data by month
- Monthly summary (Income, Expense, Profit)

---

### 📈 Charts & Visualization
- Expense charts
- Income vs Expense comparison
- Dynamic updates using Chart.js

---

### 🖥️ Dashboard UI
- Clean and simple interface
- Sections:
  - Income
  - Expenses
  - Profit
  - Charts
- Responsive design

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Storage:** JSON file  
- **Charts:** Chart.js  

---

## ⚙️ Setup Instructions

### 1. Clone Repository
```

git clone <repo-link>
cd finance-tracker

```

### 2. Install Dependencies
```

npm install

```

### 3. Run Server
```

node server.js

```

### 4. Open in Browser
```

[http://localhost:3000/login.html](http://localhost:3000/login.html)

```

## 🧠 How It Works

- Users register and log in  
- Data is stored in a JSON file  
- Expenses and income are tracked per user  
- Profit is calculated automatically  
- Charts update in real-time
